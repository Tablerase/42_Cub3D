- Alt : Slide with movements insted of turning

##### Actions

- Ctrl : fire (in Wolfenstein)
- Space : 
    - Open doors / hidden rooms (in Wolfenstein)
    - Activate elevators (in Wolfenstein)
- [1 - 4] number keys :
    - Change weapon (in Wolfenstein)
    - Change item (in Wolfenstein)
